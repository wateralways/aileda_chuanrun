六、关键禁忌

❌ 川润股份绝对不能做的事：
1. 设置3-5%的止损 → 会让胜率从100%暴跌到33%，收益从+60%降到+3%
2. 持股少于5天就卖 → 前3-5天可能浮亏或微利，第7天后才开始发力
3. 突破当天没买，次日追高 → 策略要求收盘前确认信号后买入
❌ 爱乐达绝对不能做的事：
1. 追涨突破 → 放量突破策略在爱乐达上亏损-2%，5月11日追高亏18.77%
2. 持股1-2天就卖 → 次日极少立刻大涨，必须拿满8天
3. 在RSI>70时买入 → 高位反弹陷阱
❌ 高澜股份绝对不能做的事：
1. 跳空>5%时追高 → 高开过多易回落，严格控制在0.8%~5%区间
2. 缩量跳空买入 → 必须放量确认，否则假突破概率大增
3. 持股少于5天 → 跳空高开后常有小回调，5天内卖出易亏损

七、实盘操作SOP

川润股份 SOP

1. 每天14:30开始观察
2. 检查当日是否满足放量突破四条件
3. 14:45确认信号后，收盘前买入
4. 买入后设日历提醒，8个交易日后卖出
5. 期间不盯盘，不止损，让利润奔跑

爱乐达 SOP

1. 每天14:30开始观察
2. 检查是否连阴后出现首根阳线
3. 确认RSI14 < 60
4. 14:45确认信号后，收盘前买入
5. 买入后设日历提醒，8个交易日后卖出
6. 如浮亏超4%，次日开盘考虑止损

高澜股份 SOP

1. 每天14:30开始观察
2. 检查是否跳空高开0.8%~5%
3. 确认收阳且放量（量比>1.2）
4. 14:45确认信号后，收盘前买入
5. 买入后设日历提醒，8个交易日后卖出
6. 可设6%止损+15%止盈，降低回撤
